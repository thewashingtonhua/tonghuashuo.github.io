# The Bookmark

## 简介

多年下来收藏了好多的网页，尽管都分门别类整理到文件夹了，但用到的时候还是经常会找不到。于是尝试用 Chrome 的书签管理器进行一番整理，但发现功能太基础了，不够用，找了一圈 Extension Store 也没发现自己特别中意了。没办法只好自己做一个。
