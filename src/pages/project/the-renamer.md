# The Renamer

## 简介

批量重命名工具，基于 Electron 开发。

之前一直在用一款叫 Renamer 的工具，但自从更新到 macOS Monterey 之后就用不了了，看了下 App Store，最后更新已经是一年前，直到现在也没看到有新版本的计划。无奈只好自己写一个。
