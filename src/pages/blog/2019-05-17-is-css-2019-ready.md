## CSS 还在更新吗？

好久没有听说关于 CSS 的更新了，CSS 还在更新吗？传说中的 CSS 4 似乎就跟《钢铁侠4》一样，传言就没断过。

## 所以 CSS 4 到底什么时候发布？

按照官方定义的标准，CSS 4 是不存在的，只有 CSS \[模块名称\] Level 4。

别说 CSS 4 了，甚至连 CSS 3 都是误读。
